## KEY METRICS TO TRACK MONTHLY

From **CFO_QUICK_REFERENCE.md** "Monthly Checklist":

```
[ ] Total users (cumulative)
[ ] DAU / MAU / WAU (engagement)
[ ] Revenue (actual money in)
[ ] Costs (actual money out)
[ ] Retention cohort (% of Month N-1 users still active)
[ ] LTV estimate (cumulative revenue per cohort)
[ ] CAC (how much it costs to get users)
[ ] Key feedback (what are users saying?)
[ ] Competitive threats (anyone else launching?)
[ ] Founder health (are you okay?)
```

**If any decline for 2 months consecutively:** Red flag - investigate
