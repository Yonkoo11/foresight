## The Anti-Playbook

**DON'T DO THESE:**

❌ Promise token airdrop before it's built
❌ Pay influencers before you have revenue
❌ Post on 10 different platforms
❌ Add features before validating core
❌ Chase viral moments, build consistency
❌ Spam DMs to cold influencers
❌ Overpromise, underdeliver

**DO THESE:**

✅ Stay vague on token ("early supporters rewarded")
✅ Start with no payment ("just testing commitment")
✅ Focus: Twitter + Referral (Month 1)
✅ Validate core before adding anything
✅ Build 4 posts/week, same time daily
✅ Personalize top 20 influencer DMs
✅ Promise what you can deliver TODAY
